Utilise les pipes

dans une application il n est pas rare d effectue une transformation d un element avant de l afficher l example le plus courrant est celui de la date en javascript la date a un format complexe alors queon veut plutot l information de la façon suivante 10/06/2020 le pipe est un element qui va gerer ses transformation

pour utiliser un pipe il faut utiliser l operateur (|) dans l example ci dessous le pipe utiliser est la date today | date

nous sommes le {{today | date }}

par defaut angular va utiliser le format americain , pour specifier le format a appliquer il faut passer des parametres au pipes . les parametres se places directement deriere le nom du pipe separer par un (:) donc cela donnera date:"MM/dd/yyyy"

nous sommes le {{today | date:"MM/dd/yyyy"}}

les pipes du framework

Angular met a disposition plusieurs pipes ,
  • le pipe date met en forme une date
  • le pipe async prend en entre une promise ou un observable est en affiche le resultat
  • uppercase ou lowercase
  • json transforme l element en json
  • currency : affiche les nombres dans un format monaitaire prend en param la monai a utiliser

creer un pipe

pour creer son propre pipe , il faut creer une classe qui implement interface pipeTransform , elle contient une seul methode avec la signature transform(value:any, ... args:any[]):any il faut implementer le premier param est la valeur sur laquelle est appliquer l operateur pipe | , les suivant sont ceux passe en parametre lors de l appel au pipe le retour de cette methode est le resultat du pipe on peut ajouter le pipe directement dans le composant pipes: [pipes]
